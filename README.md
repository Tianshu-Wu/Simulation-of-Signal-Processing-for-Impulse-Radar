# Simulation-of-Signal-Processing-for-Impulse-Radar
2025西工大雷达信号处理课程作业“脉冲体质雷达信号处理全流程仿真”，仅供参考，切勿抄袭






#### 1. **雷达参数设计**
- 工作频率：1GHz
- 脉冲重复间隔（PRT）：800 μs → 最大不模糊距离 120 km
- 脉冲宽度（TW）：160 μs
- 带宽（BW）：1 MHz → 脉冲压缩后距离分辨率 150 m
- 中频频率：60 MHz，采样率：80 MHz（满足带通采样定理）
- 目标设置：三个目标（25 km/50 m/s、20 km/0 m/s、20 km/30 m/s）

#### 2. **全链路仿真模块实现**
| 模块 | 功能说明 | 关键技术 |
|------|----------|----------|
| 线性调频波形产生 | 生成LFM信号 | 时宽带宽积 = 160 |
| 回波信号生成 | 模拟目标回波，加入多普勒与噪声 | 高斯白噪声，SNR = -10 dB |
| 数字中频正交采样 | 下变频至基带，获取I/Q信号 | 混频 + 低通滤波 |
| 脉冲压缩 | 匹配滤波，提升距离分辨率 | 频域卷积，FFT加速 |
| MTI/MTD | 抑制静止杂波，提取速度信息 | 脉冲对消 + 多普勒FFT |
| CA-CFAR检测 | 自适应阈值目标检测 | 保护单元 + 参考单元平均 |

#### 3. **仿真结果与验证**
- 脉冲压缩后距离分辨率达到理论值150 m。
- MTI有效抑制静止目标，MTD清晰提取出30 m/s和50 m/s的运动目标。
- CA-CFAR在低信噪比环境下实现稳定检测，虚警率可控。
